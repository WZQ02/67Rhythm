# FAQ：关于自制谱

**是的，这个音游是支持玩家自制谱面的。**
你可以点击网页右上角的上传按钮上传你编写的自制谱。

### 怎么没看到谱面编辑器？

目前这个游戏还没有做图形化的谱面编辑器，你可以用任何文本编辑软件（比如 Windows 自带的记事本）来编写谱面。
67 谱面的文本格式如下：

```
id 0 // 必填项。整数数字，自制谱可以乱填，但一定要有
title 歌曲名 // 必填项
artist 艺术家 // 可选
level 等级名 // 必填项，但可以随便填
music 音频文件网址 // 必填项，如果留空的话，上传时会要求你单独上传音频文件
cover 封面文件网址 // 可选
des 谱师名称 // 可选

150bpm // 每分钟节拍数
4 6,,,,7,,,,6,,,,7,,,,6,7,6,7,6,7,6,7, // 第一个“4”表示每小节拍子数，后面的“6,”和“7,”表示 6 音符和 7 音符，单个逗号表示空音符
300bpm // 每分钟节拍数可以中途更改
2 6,,,,7,,,,6,,,,7,,,,6,7,6,7,6,7,6,7,
4 6,,,,7,,,,6,,,,7,,,,6,7,6,7,6,7,6,7,
```

### 我导入了自制谱，music 一项填写了第三方网站上的音乐，为什么进入关卡时没声音？

大概率是被浏览器的跨域策略（CORS）限制住了。本游戏为了尽可能降低音频延迟，采用了 Web Audio API，但是这要求所有的媒体资源都能被跨域请求。大部分第三方网站是不允许这么做的。<br>
如果遇到这个问题，你可以在谱面 music 一行内容留空，然后自己上传音频文件游玩。

### 你好我是 wmc，这个谱面格式怎么看着有点眼熟？

没错，67 谱面的格式照抄了 simai（maimai 自制谱的文本格式），只不过比起 simai 更加简化了。

### 为什么官网上有几个谱面写的是 maimai 的谱师？

你可能已经猜到了，那几个谱是从 maimai 上面的谱直接转换过来的，放在网站上是作为示例。

### 也就是说……？

没错，你还可以把 simai 格式的谱面直接导进来————网站已经内置了将 simai 谱面直接转换成 67 谱面的向导，目前支持转换 tap、hold 和 touch 音符，当然，音符的方位和 hold 的长度都会被丢弃。
